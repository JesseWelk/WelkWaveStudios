WhiteStone
Designer and Developer
The latest project I have worked on is WhiteStone. The goal of WhiteStone was to create an affordable alternative to in person oral care and consultation using machine learning. Providing oral health diagnosis and recommendations for those with a lower income. WhiteStone was created with the premise of "how can we create a service with Ai to help those in need?"
This Project required a number of skills to be executed. My main role in the project came in the designing and front end implementation aspects of the app. This includes the use of figma for prototyping as well as adobe suite softwares to develop the style of the app.
In November 2023 the project was presented at the BCIT Telus theatre to the mayor of Burnaby as well as other esteemed guests.
Ecoture
Designer and Developer
An earlier project I was part of was an online interactive sustainability test called Ecoture. The point of this project was to allow visitors to test how large their carbon footprint is depending on a number of different varibles.
Ecoture required a large amount of front end design which was the aspect of the project I was assigned to. Using figma to prototype a number of the early designs as well as the adobe suite softwares to create design assets for the project.
Finally. This project required a number of different user tests to gaige the responses people might feel when using the app, The end result provided the most user friendly and welcoming UI/UX for the project.
